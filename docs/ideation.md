## 시간 슬라이더 (하단 드래그 + Snap-back)

- **출처**: `docs/product-direction.md` PRD 초안, "실시간 타임랩스" 섹션
- **원래 아이디어**: 화면 하단에 슬라이더를 두고, 유저가 드래그하면 월드맵 하늘이 그 시각(새벽/주간/황혼/야간)으로 미리보기되다가, 손을 떼면 다시 실제 현재 시각으로 자동 복귀(스냅백)하는 인터랙션.
- **현재 상태**: 미구현. 지금은 실제 현재 시각 기준으로만 하늘이 1분마다 자동 갱신됨(`WorldMap.tsx`).
- **결정 (2026.07.12)**: 반영하지 않기로 함. 실제 개발 진행 목록(`docs/progress.md`)에서 제외.

## 소장 중인 책 — 산기슭 텐트 점등

- **출처**: `docs/사용법.md` 초안(2026.07.11 기준)에 "소장 중인 책은 산기슭 텐트에 불이 켜집니다"로 기술돼 있었음.
- **원래 아이디어**: `owned`(소장 여부) 필드를 월드맵에서도 시각적으로 구분 — 산기슭에 텐트를 두고, 소장 중인 책이면 불이 켜진 것처럼 표현.
- **현재 상태**: 미구현. `components/worldmap/` 어디에도 `owned` 값을 참조해 텐트나 불빛을 그리는 코드가 없음(2026.07.12 코드 대조로 확인). 모닥불(캠프파이어)은 있지만 "읽는 중 + 야간" 또는 완등기록(trophy) 모드 조건으로만 켜지고 소장 여부와는 무관함. 소장 여부는 현재 책 카드의 집 모양 아이콘 토글로만 표현됨.
- **결정 (2026.07.12)**: 실제로 구현된 적 없는 설명이 사용법 문서에 잘못 남아있던 것으로 판단, 문서에서 제거하고 여기로 옮김. 나중에 실제로 만들지 여부는 미정.
