# 기획 방향 (Product Direction)

> 문서 출처: PRD v1.1 (2026.06.26)

---

## 핵심 컨셉

**"나의 독서 역사가 대자연의 스카이라인이 된다"**

책 한 권 = 픽셀 산 하나. 읽을수록 캐릭터가 정상을 향해 오른다. 책의 두께가 산의 높이가 되고, KDC 분류가 산의 색상을 결정하며, 유저의 독서 이력 전체가 2D 픽셀 산맥으로 가시화된다. 별도의 통계 수치나 텍스트 없이도 유저의 지적 취향이 산맥의 형태로 자동 증명된다.

---

## 공간 설계 원칙

### 가로축 — 등록 시간순 타임라인
- 책은 등록된 시간순(오래된 순 → 최신 순, Left-to-Right)으로 **고정** 배치된다
- 유저가 임의로 순서를 바꿀 수 없다 — 독서 역사의 진정성을 유지하기 위함
- 결과적으로 왼쪽 = 과거, 오른쪽 = 현재가 되어 유저의 독서 연대기가 자연스럽게 형성됨

### 세로축 — 페이지 수 기반 고도
- 총 페이지 수 → 산 높이 결정 (1단계~4단계)
- 현재 읽은 페이지 수 → 캐릭터 위치 (산기슭~정상)
- 페이지 업데이트 → 캐릭터 실시간 수직 이동

### Z축 — 팝업북 레이어 시스템
| 레이어 | 대상 | 시각 처리 |
|--------|------|---------|
| 전경 (Foreground) | 읽는 중 (1%~99%) | 100% 불투명, 100% 크기, 드롭 섀도 |
| 원경 (Background) | 미시작(0%) + 완독(100%) | 60% 투명도, 0.8배 크기 (안개 효과) |

### 실시간 타임랩스
디바이스 시각과 실시간 동기화. 시간대별 하늘 색상이 자동 변화.
- 새벽 00:00~06:00: 딥블루, 별빛 이펙트
- 주간 06:00~16:00: 아케이드풍 하늘색
- 황혼 16:00~19:00: 주홍색 노을 그라데이션
- 야간 19:00~24:00: 네이비 블루 밤하늘

**시간 슬라이더 인터랙션**: 유저가 드래그해서 시간대 미리보기 가능 → 손 떼면 현재 실제 시각으로 스냅백(Snap-back)

## 도서 등록 파이프라인

### 국립중앙도서관 API 연동 (완료)
국립중앙도서관 서지정보(SEOJI) API + Open Library API(페이지 수 보완용 폴백) 사용 중. 단, KDC 필드는 아직 응답 매핑에서 추출하지 않아 산 색상 자동 배정에는 반영되지 않음(`docs/verification.md` 참고).

연동할 데이터 6가지:

| 데이터 | 활용 | 비고 |
|--------|------|------|
| 책 제목 | 툴팁, 진도 모달, 정상석 각인 | 필수 노출 |
| 저자 | 도서 확인 팝업 | 필수 노출 |
| 총 쪽수 | 산 높이 결정, 진도율 분모 | 누락 시 300쪽 기본값 |
| KDC 첫 글자 | 산 색상 테마 자동 배정 | **유저에게 텍스트 비노출 (은닉)** |
| 발행자(출판사) | 중복 식별, 데이터 백업 | UI 비노출 |
| 발행연도 | 데이터 백업 | UI 비노출 |

### 예외 처리 규칙
- **총 쪽수 누락**: 300쪽(2단계 고도) 기본값 배정 + 유저 직접 수정 입력박스 제공
- **중복 ISBN 등록 시도**: 화면 Shake 모션 + 픽셀 말풍선 `[이미 등산로가 개척된 산입니다!]` + 강제 차단

## 진도 입력 인터랙션

### 병렬독서 모션 캐스팅
- 읽는 중(1%~99%)인 책이 여러 권이면 캐릭터가 동시에 모든 산 위에 활성화
- 캐릭터는 2비트 바운스 애니메이션(위아래로 통통 튀기)을 상시 구동

### 수직형 진도 입력 모달
- 전경 산을 터치하면 수직 모달 열림
- 페이지 입력 → 수직 게이지 바가 바닥부터 충전되며 캐릭터 실시간 슬라이딩

---

## 완등(완독) 시스템

### 8비트 완등 세레모니 (진도율 100% 도달 시)
1. 화면 중앙에 레트로 `CLEAR!` 타이포그래피 + 폭죽 파티클
2. 캐릭터가 정상에서 양팔 번쩍 + 8비트 댄스 루프 애니메이션
3. 픽셀 **정상석(바위)** 오브젝트 솟아오름 — 책 제목 + 완등일 음각
4. 유저 고유 깃발 배치

### 정상석 컬렉션 갤러리 (마이페이지)
- 완독한 책의 정상석만 바둑판(Grid) 구조로 나열
- 정상석 표면: **책 제목 + 완등일(YYYY.MM.DD)** 두 가지만 픽셀 서체로 각인
- 정상석 색상: 해당 책의 KDC 산맥 컬러 상속 (문학 → 핑크 대리석, 과학 → 민트 이끼)
- 멀리서 봤을 때 유저의 독서 취향이 컬러 스펙트럼으로 드러남

## 바이럴 캡처 시스템

- **정사각형 컬렉션 저장**: 정상석 갤러리 전체를 1:1 PNG 렌더링 → 인스타그램 피드용
- **와이드 파노라마 캡처**: 전체 산맥 타임라인을 가로로 이어 붙인 고해상도 이미지
  - 우측 하단 워터마크: `[ Read & Ridge | Climber. 닉네임 ]`
  - 인스타그램 스토리 / 노션 / 블로그 커버 활용 유도

## 비즈니스 모델

배너 광고 및 강제 영상 광고 **전면 배제**. 수집욕과 기록 감성 기반 유료화.

### Free 플랜 (무료)
- 동시 활성화(병렬독서) 최대 3권
- 연간 등록 최대 24권
- CSV 내보내기: 최근 완독 10개만 노출

### Premium Ridge 패스 (₩4,900/월)
- 병렬독서 도서 수 무제한
- 연간 도서 등록 무제한
- 전체 CSV 무제한 추출
- 구독자 한정 월간 한정판 픽셀 정상석 배지 지급

### Pixel Item Shop (부분유료화)
- 캐릭터 스킨 (탐험가, 우주비행사, 판타지 위저드, 동물 크루 등)
- 베이스캠프 아이템 (모닥불 색상, 텐트 외형, 깃발 문양)
- 정상석 갤러리 테마 (그리스 신전, 사이버펑크 네온, 오크나무 서재 등)

## 데이터 정책 (요약)

- 수집: 이메일 + 소셜 로그인 토큰 + 닉네임만
- 암호화: BCrypt / PBKDF2_SHA256 + TLS v1.3
- 휴면 1년 초과 시 개인정보 분리 보관 + 이메일 사전 통지
- 회원 탈퇴 시 Hard Delete (모든 데이터 즉시 파기)

### CSV 내보내기 필드 명세
```
"도서명","저자","출판사","발행연도","KDC분류번호","총고도(쪽수)","현재고도(읽은쪽수)","등반진행률(%)","정상정복일(완독일)"
```
