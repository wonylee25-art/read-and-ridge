# 산책또산책 (Read & Ridge) 사용법

> 이 문서는 `read-and-ridge` 프로젝트의 현재 코드를 근거로 작성한 초안입니다(최초 작성 2026.07.11 기준 / 실제 코드 대조 후 2026.07.12 정정·보강). 화면 문구·동작은 실제 앱과 동일하게 확인 후 반영했으며, 이후 기능이 바뀌면 이 문서도 함께 업데이트가 필요합니다. 문서에는 있었지만 실제로 구현된 적 없는 내용은 `docs/ideation.md`로 옮겼습니다.

---

## 1. 산책또산책이란

**산책또산책**(영문명: Read & Ridge)은 내가 읽는 책을 픽셀 아트 산으로 시각화하는 독서 기록 웹앱입니다.

- 책 한 권 = 산 하나. 책의 두께(총 페이지 수)가 산의 크기를 결정합니다.
- 읽은 만큼 캐릭터(마스코트 "마루")가 산을 오릅니다. 페이지를 업데이트하면 실시간으로 위치가 올라갑니다.
- 완독하는 순간 산 정상에서 폭죽이 터지고 캐릭터가 양팔을 들고 춤추는 완등 세레모니가 재생됩니다. 이후 정상에 깃발이 잠깐 남아있다가, 그 책은 완등기록으로 옮겨가 쌓입니다.
- 같은 책을 또 사는 걸 막기 위해 "소장 중" 여부를 별도로 관리하고, 이미 등록된 책을 중복 등록하려 하면 경고합니다.

---

## 2. 시작하기 — 로그인

1. 접속하면 로그인 여부와 무관하게 항상 산책기록(`/dashboard`)으로 이동합니다. 로그인 전에도 예시 지형도(산 4개짜리 데모 화면)를 볼 수 있어, 로그인 없이 먼저 어떤 서비스인지 둘러볼 수 있습니다.
2. 책 추가, 진행률 저장, CSV 내려받기, 회원 탈퇴처럼 실제 계정 데이터를 다루는 액션을 누르면 그 자리에서 바로 구글 로그인으로 넘어갑니다.
3. **"Google로 계속하기"** 버튼을 눌러 구글 계정으로 로그인합니다. (구글 소셜 로그인만 지원)
4. 로그인에 성공하면 자동으로 대시보드(산책기록 화면)로 이동합니다.
5. 로그인 중 오류가 발생하면 화면에 "로그인 중 문제가 발생했습니다. 다시 시도해 주세요." 메시지가 표시됩니다.

## 3. 화면 구성

왼쪽 사이드바(모바일에서는 상단 메뉴 버튼)에 두 개의 메뉴가 있습니다.

| 메뉴 | 이동 경로 | 내용 |
|------|----------|------|
| 산책기록 | `/dashboard` | 월드맵(전체 책 지도) + 읽는 중 / 잠시 멈춤 목록 |
| 완등기록 | `/dashboard/hikes` | 완독한 책만 모은 갤러리(정상에 깃발 꽂힌 산 지도) + 목록 |

화면 오른쪽 상단에는 두 페이지 공통으로 **"{닉네임}님, {인사말}"** 트리거가 있습니다. 클릭하면 "산책자 증표" 팝업이 열립니다(자세한 내용은 6장 참고).

사이드바 하단에는 다음 항목이 이 순서로 있습니다.

- **산책또산책에 대하여** — 앱 소개 팝업
- **로그아웃** (비로그인 상태면 "로그인"으로 표시)
- **CSV 내려받기** — 등록한 책 데이터를 CSV 파일로 내보내기
- **개인정보처리방침 / 이용약관**
- **회원 탈퇴**

맨 아래에는 저작권 표기(© 2026 산책또산책)와 앱 버전(`vX.Y.Z`)·마지막 업데이트 날짜가 항상 표시됩니다. 사이드바 제목 옆에는 아직 베타 서비스임을 나타내는 **BETA** 배지가 붙어 있습니다.

## 4. 책 등록하기

산책기록 또는 완등기록 화면 상단의 **"책 추가"** 버튼을 누르면 등록 폼이 열립니다.

### 4-1. 검색 또는 바코드 스캔으로 정보 채우기

- **제목 검색**: 입력창에 제목을 타이핑하면 잠시 후 국립중앙도서관 서지정보 검색 결과가 드롭다운으로 뜹니다. 원하는 책을 클릭하면 제목·저자·ISBN·페이지 수가 자동으로 채워집니다.
- **바코드 스캔**: "스캔" 버튼을 누르면 카메라가 열립니다. 책 뒤표지의 ISBN 바코드(978/979로 시작)를 비추면 자동으로 인식되어 정보가 채워집니다. 카메라 권한을 허용해야 동작합니다.
- 검색이나 스캔으로 정보를 찾지 못했다면 제목·저자를 직접 입력해도 등록할 수 있습니다.
- 검색 결과에서 얻은 KDC(도서 분류) 정보는 화면에 텍스트로 보이지는 않지만, 등록 시 함께 저장되어 월드맵에서 산 색상을 자동으로 정합니다(5장 참고).

### 4-2. 입력 항목

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 책 검색 또는 바코드 스캔 | 필수 | 제목 텍스트 |
| 저자 | 선택 | 검색 결과 선택 시 자동 입력, 직접 수정 가능 |
| 총 페이지 수 | 선택 | 비워두면 150쪽으로 등록됨 |
| 소장 중이에요 (체크박스) | 선택, 기본 체크됨 | 실제로 소장한 책이면 체크, 빌린 책·도서관 대출본이면 체크 해제 |
| 상태 | 필수, 기본값 "읽는 중" | 읽는 중 / 완독 / 잠시 멈춤 |
| 시작일 | 선택 | 날짜 |

### 4-3. 중복 등록 경고

이미 등록된 책(ISBN이 같거나, ISBN이 없다면 제목·저자가 같은 책)을 다시 등록하려 하면 화면이 흔들리며 **"OO, 또 산 책이 됩니다!"** 메시지가 뜨고 저장이 막힙니다. 등록을 진행하려면 다른 책을 선택하거나 제목을 다르게 입력해야 합니다.

## 5. 산책기록 화면 — 월드맵

`/dashboard`에 접속하면 등록된 책 전체가 픽셀 산맥으로 표시됩니다.

- **산의 크기**: 총 페이지 수가 많을수록 산이 큽니다(4단계).
- **산의 모양**: 책마다 산의 실루엣(뾰족/비대칭/고원/쌍봉)이 다르게 배정됩니다. 제목이 12자 이상으로 길면 폭이 가장 넓은 쌍봉 모양이 배정되어 제목이 덜 잘리고, 그 외에는 책마다 고정된 모양이 자동으로 골라집니다. 같은 분야(KDC)의 책도 모양이 서로 달라 산맥이 단조롭지 않습니다.
- **산의 색상**: 책의 KDC(도서 분류) 정보로 4가지 색 테마(청회색/황토/에메랄드/라벤더 계열) 중 하나가 자동으로 정해집니다. KDC 정보가 없으면 책마다 고정된 색이 순환 배정됩니다.
- **캐릭터 위치**: 읽은 페이지 비율만큼 산 위 캐릭터가 올라가 있습니다.
- **전경/배경 구분**: "읽는 중"인 책은 항상 또렷하게(전경) 표시됩니다. 전경 자리가 남으면(최대 4개) 방금 완독한 책(완등 세레모니 유예시간 동안)과 미시작 책 중 일부를 방문할 때마다 랜덤으로 채웁니다. 전경에 못 든 미시작 책만 흐릿하게(배경, 안개 효과) 표시되고, 완독한 책은 유예시간이 지나면 배경으로 가지 않고 지도에서 완전히 사라져 완등기록으로 옮겨갑니다.
- **하늘 색상**: 현재 시각에 따라 새벽·낮·황혼·밤으로 자동 변합니다.
- **날씨**: 한 달에 3~4번 정도 낮은 확률로 계절에 맞는 날씨가 함께 표시됩니다 — 4월은 벚꽃비, 5~11월은 비, 12~3월은 눈. 오늘 날짜를 기준으로 정해져서 하루 동안은 새로고침해도 같은 날씨가 유지됩니다.
- 월드맵 아래에는 **"내가 산 책" 권수**(등록한 책 전체, 상태 무관)와 **"발걸음 수"**(읽는 중·잠시 멈춤 책의 읽은 페이지 합) 통계가 표시됩니다.

### 진행률 기록하기

읽는 중인 책의 산을 클릭하면 진행률 입력 팝업이 뜹니다.

1. 세로 게이지를 드래그하거나 숫자 입력창에 직접 페이지 수를 입력합니다.
2. **저장** 버튼을 누르면 반영됩니다.
3. 총 페이지에 도달하면(100%) 버튼이 **"🏔 완독!"**으로 바뀌고, 저장 즉시 완독 처리됩니다.

같은 방식으로 목록의 책 카드에서도 페이지 숫자를 입력하고 "업데이트" 버튼을 눌러 진행률을 기록할 수 있습니다.

### 목록 (읽는 중 / 잠시 멈춤)

월드맵 아래에 상태별로 책이 카드 형태로 나열됩니다. 완독한 책은 이 목록에서 빠지고 완등기록으로 이동합니다.

## 6. 산책자 증표 (프로필)

화면 오른쪽 상단의 **"{닉네임}님, {인사말}"** 을 클릭하면 "산책자 증표" 팝업이 열립니다.

- **닉네임**: 입력창에서 언제든 바꾸고 저장할 수 있습니다. 기본값은 구글 계정 이름입니다.
- **인사말**: 2시간 단위로 시간대에 맞는 문구가 랜덤으로 바뀝니다(예: 새벽엔 "고요한 새벽이네요", 저녁엔 "노을이 참 예쁘네요"). 모바일 좁은 화면에서는 페이지 제목과 겹치지 않도록 닉네임만 표시됩니다.
- **산책 시작일 / 최근 산책일**: 가입일, 그리고 가장 최근에 책 상태를 바꾸거나 진행률을 저장한 날짜가 표시됩니다.
- **통계 카드 4개**: 내가 산 책 / 발걸음 수 / 완등기록(완독한 책 수) / 완등거리(km) — 산책기록·완등기록 데이터를 한 팝업에서 함께 볼 수 있습니다.

---

## 7. 책 카드에서 할 수 있는 것

각 책 카드에는 다음 기능이 있습니다.

- **상태 변경**: 카드 상단 드롭다운에서 읽는 중 / 잠시 멈춤 / 완독을 직접 바꿀 수 있습니다.
- **소장 여부 토글**: 집 모양 아이콘을 눌러 소장 중 여부를 언제든 바꿀 수 있습니다(예: 대출 → 구매).
- **저자 수정**: 저자 이름 옆 연필 아이콘으로 직접 입력·수정할 수 있습니다.
- **메모**: 메모 아이콘을 눌러 텍스트를 남기고 저장할 수 있습니다(⌘+Enter로도 저장 가능). 메모가 있는 책은 월드맵 산 위에도 오늘의 랜덤 말풍선(2~3개)으로 잠깐씩 노출됩니다.
- **페이지 업데이트**: 숫자 입력 후 "업데이트" 버튼.
  - 현재 페이지가 총 페이지 이상이 되면 자동으로 "완독" 상태로 전환됩니다.
- **삭제**: 휴지통 아이콘을 누르면 확인 팝업이 뜬 뒤 삭제됩니다(실수 방지).

## 8. 완독과 완등기록

책의 진행률이 100%에 도달하면(직접 입력 또는 상태 드롭다운에서 "완독" 선택) 다음 순서로 진행됩니다.

1. 산 정상에서 **폭죽이 터지고** "CLEAR!" 픽셀 글자가 뜨며, 캐릭터가 양팔을 들고 춤추는 완등 세레모니가 약 10초간 재생됩니다.
2. 세레모니가 끝나면 정상에 **깃발**이 꽂힌 채로 잠깐(세레모니 포함 약 10.5초) 산책기록 지도에 남아있습니다.
3. 그 시간이 지나면 산책기록 지도에서는 사라지고, 그 책은 **완등기록**(`/dashboard/hikes`) 화면으로 옮겨갑니다. (책 카드 자체에는 "완독 완료! 🎉" 배너가 잠깐 표시됩니다.)

완등기록 화면에는:

- 정상에 깃발이 꽂힌 완독 책들의 월드맵(정상석 전용 모드 — 완독한 책들을 부르는 이름이 "정상석"이며, 실제 화면에는 산과 깃발로 표시됩니다)
- 최근 완독한 5권만 지도에 표시되며, 그 이상은 아래 전체 목록에서 확인 가능
- **완등 기록 수**(완독한 책 수), **완등 거리(km)**(읽은 총 페이지를 거리로 환산한 값) 통계
- 완등기록 화면에서도 상단 "책 추가"로 바로 새 책을 등록할 수 있습니다.

## 9. 화면 저장하기 (PNG 캡처)

두 화면 모두 왼쪽 하단에 카메라 아이콘 버튼이 있습니다(등록한 책이 있을 때만 노출).

- **완등기록(`/dashboard/hikes`)에서 누르면**: 완독한 책만 모은 "완독 맵" 파노라마를 PNG로 즉시 저장합니다. 최근 5권은 선명하게, 그 뒤는 흐릿한 능선으로 표시되고 각 산에 책 제목이 각인됩니다. 파일명: `산책또산책_완독맵_날짜.png`
- **산책기록(`/dashboard`)에서 평소에 누르면**: 지금 화면에 보이는 월드맵 그대로(읽는 중·완독·배경 산·날씨까지)를 PNG로 저장합니다. 파일명: `산책또산책_산책기록월드맵_날짜.png`
- **방금 완독한 책이 있을 때 산책기록에서 누르면**: 버튼이 호박색으로 강조되며 "인증샷 찍기" 슬라이드 버튼으로 바뀝니다. 누르면 방금 완독한 산을 중심에 크게, 다른 책들을 옆에 작게 곁들인 정사각형 "정상 인증샷"을 PNG로 저장합니다(폭죽·CLEAR!·춤추는 캐릭터, 왼쪽 상단엔 KDC 색 구성 뱃지, 오른쪽 하단엔 닉네임 워터마크 포함). 파일명: `산책또산책_정상인증샷_날짜.png`

## 10. 데이터 내보내기 / 계정 관리

- **CSV 내려받기**: 사이드바의 "CSV 내려받기"를 누르면 등록한 책 데이터를 CSV 파일로 받을 수 있습니다.
- **로그아웃**: 사이드바 하단 로그아웃 버튼.
- **회원 탈퇴**: 사이드바 하단 "회원 탈퇴" 클릭 → 확인 팝업에서 **"탈퇴합니다"**를 정확히 입력해야 버튼이 활성화됩니다. 탈퇴 시 등록한 모든 책·독서 진행률·완독 기록과 계정이 즉시 삭제되며 되돌릴 수 없습니다.

## 11. 숨은 요소

앱 안에 사용자에게 따로 안내되지 않는 히든 기능이 하나 있습니다.

- **오로라 이스터에그**: 개발자가 미리 지정해 둔 특정 ISBN의 책을 등록하면, 화면 전체에 10초짜리 초록-보라 오로라 연출이 1회 재생됩니다. 그 책은 이후 완독해도 깃발 색이 평범한 6색 팔레트 대신 오로라 팔레트(초록-보라 계열)로 고정됩니다. 어떤 책인지는 의도적으로 공개하지 않는 개인적인 발견 요소입니다.

## 12. 확인이 더 필요한 부분

- Premium 유료 플랜, 캐릭터 스킨 등 상점 기능은 기획 문서(`docs/product-direction.md`)에만 있고, 실제로는 아직 구현되지 않았습니다. 유료 정책 자체가 아직 결정되지 않은 상태입니다(`docs/progress.md` "Phase 8" 참고).
- 완등 거리(km) 환산은 코드상 "페이지당 0.225m"로 계산됩니다. 신국판(152×225mm) 판형을 기준으로 1쪽 = 225mm로 확정한 값입니다(`docs/progress.md` 2026.07.08 기록 참고).
