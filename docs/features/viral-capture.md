## 컨셉

공유 버튼을 일반적인 "공유하기"가 아니라 **"정상 인증샷 찍기"**로 프레이밍한다. 실제 등산 문화의 정상석 인증샷 의식을 앱 안으로 그대로 가져오는 것이 목적 — 단순 스크린샷이 아니라 "내가 이 산에 올랐다"는 서사가 담긴 결과물이어야 한다.

캡처는 페이지 맥락(산책기록 / 완등기록)에 따라 **서로 다른 두 종류**로 나뉜다. `WorldMap.tsx`가 이미 `mode='home'`(산책기록)과 `mode='trophy'`(완등기록)로 렌더링을 분기하고 있으므로, 캡처 로직도 이 기존 분기 위에 얹는다.

---

## 두 가지 캡처 모드

| | 산책기록 캡처 (`mode='home'`) | 완등기록 캡처 (`mode='trophy'`) |
|---|---|---|
| 트리거 시점 | 완독 세레모니 직후 — 그 산이 아직 전경에 남아있는 유예시간(`COMPLETION_GRACE_MS`) 동안 | 페이지 방문 중 유저가 원할 때 버튼으로 |
| 화면 구성 | 방금 완독한 산이 전경 중심(세레모니 이펙트: CLEAR! 타이포 + 폭죽 + 정상 댄스 캐릭터 포함), 나머지 산은 기존 Z레이어 규칙대로 배경 안개(opacity 0.5, scale 0.82) | trophy 모드가 이미 그리는 화면 그대로 — 완독 산들이 나열된 산맥, 캐릭터 없음, 항상 맑음+모닥불 |
| 성격 | 순간 이벤트 — "지금 이 산 정상에 도착했다" | 누적 컬렉션 — "그동안 내가 오른 산들 전부" |
| 포맷 | 정사각형(1:1) — 인스타 피드용 | 가로 파노라마 — 인스타 스토리/블로그 커버용 |
| 문구 톤 예시 | "OO산 정상 도착!" | "지금까지 오른 산들" |

## 왼쪽 상단 컬러 뱃지 (KDC 색 구성 요약) — ✅ 구현 완료 (`drawKdcBadge`/`countByTheme`)

산 색 스와치(mystery/earth/nature/fantasy 4색) + 개수만 표시하는 픽셀 뱃지. `design-style.md`의 "KDC 값은 텍스트로 절대 노출하지 않는다" 원칙을 지켜 라벨(예: "문학 5권") 없이 **색+숫자만**(예: 🟣5 🟢2 🟠4 🔵3) 사용한다. 0권인 색은 뱃지 자체를 생략한다.

집계 기준은 모드별로 다르다:

- **산책기록 캡처**: 책장 전체(읽는 중 + 완독 + 미시작 포함) 기준 → "내 전반적인 취향"을 보여주는 용도
- **완등기록 캡처**: 완독한 산만 기준 → "내가 실제로 정복한 것들"을 보여주는 용도

### ⚠️ 주의: trophy 화면 표시 캡과 뱃지 숫자는 분리해야 함

`worldmap-utils.ts`의 trophy 모드는 화면 복잡도 관리를 위해 최근 완독 N권(`TARGET_TROPHY`)까지만 전경에 그리고 나머지는 잘라낸다. 완등기록 캡처의 뱃지 숫자를 "화면에 그려진 것"만으로 세면, 책을 많이 읽은 유저일수록 오히려 숫자가 작게 나오는 역설이 생긴다.

→ **화면에 그리는 산은 캡(cap)에 걸려도 되지만, 뱃지에 적히는 숫자는 캡과 무관하게 실제 누적 완독 전체 개수를 별도로 집계해서 사용한다.**

✅ 반영됨: `renderCompletedPanorama(completedBooks, ...)`는 화면에 그릴 `front`(=`completedBooks.slice(0, TARGET_TROPHY)`)와 뱃지 집계용 `completedBooks`(캡 없는 전체 목록, 호출 측이 이미 넘겨줌)를 분리해서 쓴다.

## 공통 디자인 원칙

캡처 결과물이 "예쁘다"의 기준을 아래 다섯 축으로 명세한다 (감이 아니라 체크 가능한 기준으로):

1. **브랜드 일관성** — 새 스타일을 만들지 않고 `design-style.md`에 이미 정의된 KDC 파스텔 팔레트·시간대 하늘 그라데이션을 캡처 시점 그대로 반영한다.
2. **정보 절제** — 진행률 %, 페이지 수 같은 수치를 나열하지 않는다. 책 제목 + 완독일 정도만 최소 노출(`design-style.md`의 "텍스트 정보 밀도 최소화" 원칙).
3. **포맷 적합성** — 용도에 맞는 별도 구도(정사각형 vs 파노라마)를 각각 렌더링한다. 하나의 캔버스를 억지로 재사용하지 않는다.
4. **순간성** — 정적 스냅샷보다 세레모니 순간(폭죽 피크 프레임 등 타이밍 지정 필요)을 캡처해 서사를 담는다.
5. **고유성** — 산 실루엣 다양화(제목 길이 + 책 id 시드 기반, **완료 2026.07.12** — 아래 선행 조건 참고)가 들어가야 "내 산맥"이 남과 달라 보여 캡처의 자랑거리가 커진다.
