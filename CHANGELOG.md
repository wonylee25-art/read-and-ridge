# Changelog

산책또산책(Read & Ridge)의 버전별 변경 이력. 배포(push)할 때마다 여기에 기록하고,
`package.json`의 `version`과 `lib/version.ts`의 `LAST_UPDATED`도 같이 갱신할 것.

## 0.2.0 — 2026-07-11

- **책 추가 팝업**: 검색 결과 목록이 길어져도 팝업이 늘어나지 않고 목록 안에서만
  스크롤되도록 수정 (`max-h-72 overflow-y-auto`).
- **책 추가 팝업**: 저자가 여러 명이라 문자열이 아주 길어지는 경우, 자동 입력되는
  저자 값을 30자로 자르도록 수정 (`truncateAuthors`). 팝업 크기가 항상 고정되도록.
- **버전 표기**: 사이드바에 BETA 배지, 앱 버전(`vX.Y.Z`), 마지막 업데이트 날짜 표시 추가.
  버전 관리 체계 도입(`lib/version.ts`, 이 CHANGELOG).

## 0.1.0 — 2026-07-09

- 베타 오픈: 구글 로그인, RLS, Vercel 환경변수 설정 완료 후 최초 배포.
- 팝업 위치/레이아웃 버그 4건 수정 (자세한 내용은 `배포전_체크리스트.md` 참고).
